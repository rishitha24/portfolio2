Projects

Image Captioning

1. One of the main challenges faced by Computer Vision is Scene Understanding! This project is an idea to generate a caption describing the picture in a sentence.
2. Model was built using the pretrained RESNET50 Architecture, LSTM's. Vocabulary was extracted from the captions that were available in the train datasets using Lemmatation from nltk in python.

Smart Green House

1. The aim of this project is to monitor and control a green house with sensors and actuators through a mobile app connected to the internet dynamically.
2. Sensors that were used are soil moisture, Temperature and Humidity - DHT 11 and LDR for light sensor.
3. Data was collected from the sensors through a microcontroller Arduino here, and stored in the firebase.
4. The web application and the mobile app was built using HTML5, CSS and very few bootstrap features!

Github link for the Project

Multiple Motor Synchronization

1. This project is an idea to replace huge mechanical devices like gears with an embedded device (tried with Arduino) to synchronize many motors in industries.
2. Using an embedded device would be benificial because we can over came wear & tear of machines also labour work can be reduced.
3. The technique that was used to synchronize motors was "Adjacent cross-coupling" keeping delay in the mind.

Spatial Modulation

1. This project is an idea to simulate Spatial Modualtion technique with Space Time Block Coding (STBC) to achieve higher data rate, spectral efficiency and reliability.
2. STBC is chosen over Spatial Multiplexing (SM) because, SM using V-Blast requires complex decoders at the receivers as they are prone to high channel-interface.
